## Tentang Saya
Saya Sherif, fokus di keamanan siber, pengembangan, dan infrastruktur. Saya biasa menjembatani perspektif ofensif (red teaming, pentest) dan defensif (hardening, monitoring) agar setiap solusi tetap aman, cepat, dan mudah dirawat.

## Keahlian Utama
- **Cyber Security:** Web/API pentest, secure code review, threat modeling, hardening Linux, SIEM logging, incident response dasar.
- **Development:** Backend (Node.js/TypeScript, Python), REST/GraphQL API design, automation scripting, CLI tooling, testing dan CI/CD dasar.
- **Infra & Ops:** Linux server, container dasar, reverse proxy (Nginx/Caddy), monitoring ringan (Prometheus stack), cloud dasar (VPS/konten statis).

## Cara Saya Bekerja
- Security by design: linting, SAST ringan, dependency audit, prinsip least privilege sejak awal.
- Dokumentasi ringkas agar operasional dan handover tetap mulus.
- Automasi tugas berulang dengan script atau pipeline sederhana.

## Highlight Singkat
- Mengurangi temuan OWASP Top 10 lewat secure coding checklist dan sesi review.
- Membangun API dengan rate limiting, audit logging, dan alert sederhana untuk anomali.
- Menyiapkan playbook respons insiden ringan agar tim non-security bisa bertindak cepat.
